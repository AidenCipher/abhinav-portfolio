# Personal Branding Website

A minimal, intellectual personal branding website built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

**IMPORTANT: Install dependencies first!**

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Troubleshooting

If you see errors about `framer-motion` not being found, run:
```bash
npm install
```

This will install all required dependencies including `framer-motion`.

## Project Structure

- `/app` - Next.js App Router pages and layouts
- `/app/api` - API routes (contact form)
- `/components` - Reusable components
- `/lib` - Data and utilities
- `/public` - Static assets

## Adding Writing

Edit `/lib/writing.ts` to add or update curated writing links.

## Deployment

This site is configured for Vercel deployment. Connect your repository and deploy.
